Ecosystem Overview

The SYNTROPY ecosystem covers the full forensic lifecycle:

1 Audit — Kernel hardening assessment (LinSpec)
2 Acquire — Forensic memory acquisition (SIREN)
3 Analyze — RWX memory scanning & detection (K-Scanner)
4 Validate — Adversarial scenario validation (K-Verify)

Architecture

The pipeline flows: LinSpec (audit) → SIREN (acquire) → K-Scanner (analyze) → K-Verify (validate). SYSCAGE provides complementary syscall profiling and confinement. PMV extends the ecosystem to OpenBSD.

All tools produce structured output (report.json) consumable by the next stage in the pipeline.
